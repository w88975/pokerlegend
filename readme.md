PokerLegen React
----

- 启动服务
```shell
npm install webpack -g // 全局安装`webpack`包
npm install . // 安装`node`依赖
webpack // build项目
node app.js // 开启http server
```
    - 浏览器中预览: [http://localhost:9999](http://localhost:9999)

- Webpack 参数
```shell
webpack --watch // 开启`webpack`的`watch`操作
webpack --p // build项目的时候选择压缩
```
